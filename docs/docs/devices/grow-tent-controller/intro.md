# Introduction

The Grow Tent Controller serves as a device that can monitor and maintain the
environment inside of a grow tent and integrate with the Agrigate platform.

The main functions of this device are to

- Capture the temperature and humidity of the grow tent
- Maintain a lighting schedule
- React to fluctuations in temperature and humidity, keeping them within a
  pre-defined range
- Store telemetry in a database for reporting and analysis
- Provide a touch-screen interface for control
- Allow access and control from a mobile or desktop device via Agrigate
