# 0.1.0 - 2024-10-13

## Summary

Initial release of the client application for interacting with the Agrigate
platform. Allows users to specify a server URL, add devices, and view devices or
telemetry.

## Features

- Users can specify the Agrigate server URL
- Users can view a list of devices and add a new device
- Users can view the details and telemetry of a particular device
