# 0.1.0 - 2024-09-29

## Summary

The initial release of the Agrigate Event Service. Connects to a particular
topic on an MQTT broker in order to retrieve telemetry readings from devices.

## Features

- Subscribes to a telemetry channel on an MQTT Broker and saves incoming
  telemetry to the Agrigate database
